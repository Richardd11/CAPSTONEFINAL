## 🚀 How to Use

### **1. Update HTML Script Tags**

**For Exam Builder:**
```html
<!-- Core -->
<script src="/js/core/api-client.js"></script>

<!-- Utils -->
<script src="/js/utils/template-engine.js"></script>

<!-- Services -->
<script src="/js/services/toast.service.js"></script>

<!-- Models -->
<script src="/js/models/question.model.js"></script>
<script src="/js/models/exam.model.js"></script>

<!-- Views -->
<script src="/js/views/exam-builder.view.js"></script>

<!-- Services -->
<script src="/js/services/exam.service.js"></script>

<!-- Controllers -->
<script src="/js/controllers/exam-builder.controller.js"></script>

<!-- App -->
<script src="/js/app/exam-builder.app.js"></script>
```

**For Admin Dashboard:**
```html
<!-- Core -->
<script src="/js/core/api-client.js"></script>

<!-- Services -->
<script src="/js/services/toast.service.js"></script>

<!-- Models -->
<script src="/js/models/user.model.js"></script>

<!-- Views -->
<script src="/js/views/user-management.view.js"></script>

<!-- Services -->
<script src="/js/services/user.service.js"></script>

<!-- Controllers -->
<script src="/js/controllers/user-management.controller.js"></script>
<script src="/js/controllers/admin-dashboard.controller.js"></script>
```

### **2. Follow Load Order**
Always load in this sequence:
1. Core
2. Utils
3. Services (independent)
4. Models
5. Views
6. Services (dependent)
7. Controllers
8. App

### **3. Respect MVC Rules**
- Models: Data only
- Views: UI only
- Controllers: Coordination only
- Services: Business logic only

## 🎓 Best Practices

### **Adding New Features**

1. **Create Model** (`models/[feature].model.js`)
   ```javascript
   class FeatureModel {
       constructor(data) { /* ... */ }
       validate() { /* ... */ }
   }
   ```

2. **Create View** (`views/[feature].view.js`)
   ```javascript
   class FeatureView {
       render(data) { /* ... */ }
       update() { /* ... */ }
   }
   ```

3. **Create Service** (`services/[feature].service.js`)
   ```javascript
   class FeatureService {
       async getData() { /* ... */ }
       async saveData(data) { /* ... */ }
   }
   ```

4. **Create Controller** (`controllers/[feature].controller.js`)
   ```javascript
   class FeatureController {
       constructor() {
           this.model = new FeatureModel();
           this.view = new FeatureView();
           this.service = new FeatureService();
       }
   }
   ```

5. **Create App** (`app/[feature].app.js`)
   ```javascript
   document.addEventListener('DOMContentLoaded', () => {
       window.featureController = new FeatureController();
   });
   ```
