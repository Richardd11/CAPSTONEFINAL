Technical Details

1. Modal Scope Fix

Fixed `savingModal is not defined` error by moving variable declaration outside try block:

// Before (Error) try { let savingModal = null; // ... code ... } catch (error) { if (savingModal) savingModal.close(); // ❌ ReferenceError } // After (Fixed) let savingModal = null; // ✅ Declared in outer scope try { // ... code ... } catch (error) { if (savingModal && typeof savingModal.close === 'function') { savingModal.close(); // ✅ Works correctly } }

2. Data Format Conversion

Added format conversion methods to match backend validation requirements:

// Backend expects specific formats: year_level: "1st Year" (not "1") semester: "1st Semester" (not "1") section: "A" (already correct) // Frontend conversion: formatYearLevel("1") → "1st Year" formatSemester("1") → "1st Semester"
